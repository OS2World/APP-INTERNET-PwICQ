AIM/Oscar Protocol Specification:
Section 3: BOS Signon: Rate Request


Rate Request [Source: Client]

Sent by the client so it can know how fast it can send SNACs. If this rate is disobeyed, you'll be (at worst) disconnected.

SNAC Information:

  • Family 0x0001
  • SubType 0x0006
  • Flags {0x00, 0x00}
ContainerDataTLV Type
RAWSNAC Header
